clone repository ke viscode atau download file zip dari git.
1. jalankan npm install pada terminal.
2. jalankan server local(xammp/laragon/etc.) untuk akses ke database mysql.
3. jalankan php artisan migrate pada terminal.
4. jalankan npm run dev.
5. buka terminal baru, dan jalankan perintah php artisan serve. web akan dapat diakses melalui link local host yang diberikan. umumnya berjalan pada http://127.0.0.1:8000.
6. untuk dapat menggunakan payment gateway, membutuhkan public server yang dapat diakses oleh pihak ketiga(midtrans).
